## cc98-summary

基于Azure openai GPT-3.5的对于cc98帖子进行总结的赛博算命师

接入cc98登录中心,通过cc98-api获取帖子

需要在校园网下运行（webvpn不彳亍

❗本项目给出的总结仅供参考，不代表本人观点，也不对其真实性负责

### 技术栈

- next.js 13
- react-oidc-context
- tailwind css
- material UI
- [commit规范](https://www.conventionalcommits.org/en/v1.0.0/)
